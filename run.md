# Run

Run migrations

```bash
~/modulr-laravel$ php artisan migrate
```

Run seeders _optional_

> NOTE: Seeds create 10 users

```bash
~/modulr-laravel$ php artisan db:seed
```

Build js, css & assets

> NOTE: Developer mode

```bash
~/modulr-laravel$ npm run dev
```

Run serve

```bash
~/modulr-laravel$ php artisan serve
```
